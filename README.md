# HealthClockW

## 演示站点

- http://4dvd5y.natappfree.cc/heath/

本项目为专属于攀大学子设计的超星学习通自动健康打卡系统。

## 使用说明

1、此应用不作任何非法行为，仅用于测试

2、输入手机号以及学习通密码，单击启动自动打卡即可完成在服务器登记

3、学习通修改密码后需要重新在本页面进行登记

4、进行登记后,服务器会每天开始给登记的学习通账号自动打卡

5、年级、专业、班级以及当前所在省、市、县（区）可不填,不填则默认选择学习通上一天打卡记录中的值
